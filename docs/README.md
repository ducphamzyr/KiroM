# Screenshots

Place the following files here for the README to display correctly:

- `logo.png` — Project logo (80×80 recommended)
- `screenshot-dashboard.png` — Admin panel dashboard showing accounts + stats
- `screenshot-settings.png` — Settings tab (2-column layout, Telegram config)
- `screenshot-console.png` — Console tab (API endpoints + log panel)

**How to capture:**
1. Run `kiro-go.exe` and open `http://localhost:8080/admin`
2. Use browser DevTools (Ctrl+Shift+P → "Capture screenshot") or Snipping Tool
3. Recommended width: 1200px for best display on GitHub
